# Usage
<!-- usage -->
```sh-session
$ npm install -g konfig-manager
$ konfig COMMAND
running command...
$ konfig (-v|--version|version)
konfig-manager/1.0.0 darwin-x64 node-v10.15.1
$ konfig --help [COMMAND]
USAGE
  $ konfig COMMAND
...
```
